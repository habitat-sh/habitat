Habitat 0.71.0 Release Notes

We are happy to announce the release of Chef Habitat v0.71.0. We have a number of new features, so please read on for all the details. If you just want the binaries, head on over to
[Install Chef Habitat](https://www.habitat.sh/docs/using-habitat/#install-habitat).

As always thank you for using Chef Habitat!

Higlights:

#### New Features & Enhancements
- Add --reverse option to hab pkg dependencies [#5923](https://github.com/habitat-sh/habitat/pull/5923) ([jamesc](https://github.com/jamesc))
- Add --transitive option to hab pkg dependencies [#5940](https://github.com/habitat-sh/habitat/pull/5940) ([jamesc](https://github.com/jamesc))

#### Behavioral Changes
- Remove minimum 3 member requirement from quorum check [#5965](https://github.com/habitat-sh/habitat/pull/5965) ([baumanj](https://github.com/baumanj))
